description: Manage Cifra protected paths (the AI Privacy Shield blocklist)
argument-hint: "[add <path> | remove <path>]"
---

Manage protected paths with `cifra protect`:

- No argument → run `cifra protect list` and show the registered patterns.
- `add <path>` → run `cifra protect add <path>` to block Claude Code from reading that path.
- `remove <path>` → run `cifra protect remove <path>`.
