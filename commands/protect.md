description: Manage Envault protected paths (the AI Privacy Shield blocklist)
argument-hint: "[add <path> | remove <path>]"
---

Manage protected paths with `envault protect`:

- No argument → run `envault protect list` and show the registered patterns.
- `add <path>` → run `envault protect add <path>` to block Claude Code from reading that path.
- `remove <path>` → run `envault protect remove <path>`.
